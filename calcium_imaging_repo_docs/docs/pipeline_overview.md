# Pipeline Overview

This document summarizes the experimental and computational workflow represented by the thesis analysis pipeline.

## 1. Experimental input

The experimental input consisted of pancreatic tissue-slice calcium-imaging recordings from three genotype groups:

- WT: wild-type mice
- HET: heterozygous Ctrb1-Δex6 mice
- MUT: homozygous mutant Ctrb1-Δex6 mice

The tissue slices were loaded with the calcium indicator Calbryte 520 and imaged during sequential stimulation with glucose and acetylcholine (ACh).

## 2. Imaging-derived data

The computational workflow begins from imaging-derived outputs generated by the laboratory workflow, including:

- ROI definitions
- fluorescence traces
- detected calcium events
- experiment metadata
- stimulation protocol information
- acquisition-scope metadata

A region of interest (ROI) represents an image-defined cellular region from which one calcium-fluorescence trace is extracted and analyzed as a single observational unit.

## 3. Event and phase organization

Detected calcium events were assigned to experimental phases based on protocol timing information. Stable phase-level metrics were calculated after excluding transition periods following stimulation changes.

The main cross-genotype comparison focused on the 100 nM ACh phase.

## 4. ROI-level feature construction

For each ROI, calcium-event descriptors were summarized into analysis-ready features, including:

- baseline event rate during 8 mM glucose
- ACh event rate
- ACh mean event half-width
- latency to first ACh-associated event
- coefficient of variation of inter-event intervals during ACh stimulation

These features formed the final ROI-level feature space for clustering and comparison.

## 5. Quality control

Quality-control steps included:

- exclusion of incompatible acquisition formats from the final comparison dataset
- acquisition-scope-aware spatial filtering
- filtering of extreme technical event artifacts
- exclusion of recordings based on general quality criteria
- correction of protocol-timing issues where necessary

General exclusion criteria included severe tissue drift, absence of clean acinar ROIs, poor apparent cell viability, erratic baseline activity, very low ROI yield, or technical acquisition artifacts that could bias downstream feature extraction.

## 6. Unsupervised clustering

Gaussian Mixture Models (GMMs) were fitted separately within each genotype using the ROI-level feature space. The resulting two-component framework was used to describe Phasic and Sustained response modes as operational response descriptors, not as fixed biological cell identities.

## 7. Genotype-aware comparison

The final analysis compared response-mode proportions and feature distributions across WT, HET, and MUT groups. Comparisons were evaluated both on pooled data and within acquisition scope where relevant.

## 8. Event-composition analysis

As a complementary analysis, individual calcium events were re-examined within a fixed stable window of the 100 nM ACh phase. ROIs were categorized according to the proportion of short versus long events in this window.

## 9. Visualization and inspection

The Streamlit application was developed to support transparent inspection of the final processed dataset, model outputs, feature distributions, response-mode proportions, event-composition results, and spatial maps.

## Confidentiality

This repository documents the workflow and software structure. Raw experimental data and confidential full analysis tables are not publicly distributed.
