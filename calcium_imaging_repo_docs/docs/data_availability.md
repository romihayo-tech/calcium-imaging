# Data Availability

The raw experimental data used in this project are not publicly available.

The data were generated within an ongoing research line of the host laboratory and are subject to confidentiality restrictions. Therefore, this public repository does not include:

- raw microscope imaging files
- raw ROI pickle files
- full raw event tables
- complete ROI-level experimental tables
- animal-level or slice-level confidential metadata
- unpublished full analysis outputs not approved for public release

## What is included instead

This repository may include code, documentation, and synthetic example files that illustrate the expected input-table structure without revealing real experimental data.

Synthetic example files are provided only for demonstration and testing of repository structure. They are not derived from the experimental dataset and should not be used for biological interpretation.

## Access to real data

Access to the underlying experimental data remains restricted and may be considered only for authorized reviewers or collaborators, subject to approval by the host laboratory and applicable confidentiality requirements.
